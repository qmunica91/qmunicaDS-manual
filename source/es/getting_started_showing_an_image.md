# Mostrar una Imagen en Pantallas

Añade un archivo de **Imagen** a tu **Biblioteca Multimedia** y **Prográmalo** para reproducirse en **Pantallas**.

Es posible que tengas contenido de imagen creado fuera del Editor de Diseños, por ejemplo, creado por una agencia, equipo de marketing interno, etc., listo para mostrarse a pantalla completa en tu red de pantallas.

Los archivos de imagen se añaden primero a la Biblioteca mediante subida o URL. Para este ejemplo, se subirá un archivo de imagen a la Biblioteca Multimedia del CMS.

### Añadir Multimedia

- Haz clic en **Multimedia** bajo la sección **Biblioteca** del menú principal del CMS.
- Haz clic en el botón **Añadir Multimedia**.
- Selecciona **Añadir archivos** y elige el archivo de imagen a subir.
- Proporciona un nuevo **Nombre** o deja este campo en blanco para mantener el nombre original del archivo.
- Haz clic en **Iniciar subida**.

- Una vez subido, haz clic en **Hecho**.

{version}
Se debe tener precaución al subir imágenes muy grandes. Los usuarios deben comprobar la **Resolución** de la Imagen y mantenerla dentro del tamaño de pantalla previsto para obtener los mejores resultados.
{/version}

### Seleccionar Multimedia

- Desde la cuadrícula de **Multimedia**, usa los filtros para facilitar la búsqueda del archivo de **Imagen** subido si es necesario.
- Usa el menú de fila y selecciona **Programar**:

![Media Row Menu Schedule](img/v4_showing_images_videos_row_menu_schedule.png) 

Se abrirá el formulario **Seleccionar Multimedia** para proporcionar opciones de configuración adicionales para la Imagen:

**Duración en bucle** determina cuánto tiempo se mostrará esta Imagen cada vez que aparezca en un **Bucle de Programación**. 

- Especifica una **duración** para anular la duración del archivo de imagen.
- Selecciona una **Resolución** específica para usar.
- Establece un **color** para rellenar los huecos en el fondo si la Imagen no llena la pantalla.
- **Guardar** para cerrar el formulario **Seleccionar Multimedia** y continuar con la programación.

### Programar Evento

 Completa los campos en el formulario **Programar Evento**:

- Dale al evento un **Nombre** para facilitar su identificación en el CMS.

- Selecciona en qué **Pantallas** se mostrará esta Imagen.

- Desde **Partición del día**, selecciona **Personalizado** para establecer fechas/horas de inicio y fin específicas o elige **Siempre** para mostrar la Imagen en las Pantallas constantemente.

- Haz clic en **Guardar.**

El CMS actualizará ahora un cambio de Programación y la Imagen se mostrará en las Pantallas durante los tiempos establecidos en la programación.

## Preguntas Frecuentes

***¿Qué son los Bucles de Programación?*** 

Un Bucle de Programación es cómo se reproducirá el contenido multimedia cuando esté programado con otros contenidos multimedia. Por ejemplo, si has programado 2 elementos multimedia cada uno con una duración de 10 segundos para mostrarse al mismo tiempo durante 1 hora, ambos elementos se reproducirán durante sus duraciones de 10 segundos en rotación equitativa durante el tiempo programado.

***¿Qué extensiones de archivo son compatibles al subir?***

Extensiones de archivo válidas: jpg, jpeg, png, bmp, gif, webp.

***¿Puedo subir un gif animado?***

Los gifs animados no son compatibles.
