# Usando Plantillas

Usa **Plantillas** para acelerar el proceso de diseño de modo que los Usuarios puedan personalizar el contenido en lugar de tener que empezar desde cero construyendo diseños de Layout. Agiliza la creación de contenido con Diseños prediseñados para permitir a los Usuarios concentrarse en el contenido principal mientras mantienen los estándares visuales y pautas de la empresa/industria.

## Diseños Publicados como Plantillas

Los Diseños creados en el Editor de Diseños pueden guardarse para futura selección como una Plantilla después de Publicar:

- Desde el **Editor de Diseños**, abre el menú superior.
- Desde el menú desplegable, selecciona **Guardar Plantilla**.
- Completa todos los campos relevantes del formulario, optando por incluir todos los Widgets en el diseño de la Plantilla si es necesario.
- **Guardar**.

Los Diseños Publicados también pueden guardarse como una Plantilla desde la cuadrícula de **Diseños**:

- Navega a **Diseños** bajo la sección **Diseño** del menú principal del CMS.
- Usa el menú de fila para un Diseño Publicado y selecciona **Guardar Plantilla**.
- Completa los campos del formulario y **Guardar**.

{tip}
¡Los Diseños con la etiqueta 'Plantilla' también añadirán el Diseño a tu lista de diseños de Plantilla!
{/tip}

## Usando Plantillas para crear nuevos diseños de Layout

Crea un nuevo diseño para tu Layout usando una Plantilla desde dentro del Editor de Diseños:

- Haz clic para abrir la **Caja de Herramientas**.

- Selecciona el botón **Buscar Plantillas de Diseño**.

- Usa los filtros para reducir los resultados de búsqueda.

- Arrastra y suelta la Plantilla seleccionada al lienzo.

- Personaliza o añade elementos según sea necesario.

- Usa el menú superior para Publicar una vez satisfecho con el diseño.


{nonwhite}

## Plantillas de Xibo Exchange

Selecciona de un catálogo actualizado de diseños de Plantillas Xibo. Cada Plantilla ha sido creada utilizando estándares de diseño establecidos y puede servir como un gran punto de partida para crear contenido de aspecto profesional sin amplios conocimientos de diseño. Las Plantillas son totalmente personalizables y pueden adaptarse para satisfacer las necesidades de contenido. 

Las Plantillas Xibo están disponibles desde dentro del Editor de Diseños:

- Haz clic para abrir la Caja de Herramientas.
- Selecciona el botón Xibo Exchange.
- Usa los filtros para reducir los resultados de búsqueda.
- Arrastra y suelta la Plantilla seleccionada al lienzo.
- Personaliza / adapta según sea necesario.
- Usa el menú superior para Publicar una vez satisfecho con el diseño.

{version}
**NOTA:** Seleccionar una Plantilla reemplazará el diseño de Layout en el que estás trabajando actualmente con la Plantilla seleccionada. Esta acción es irreversible y debe usarse con precaución.
{/version}
{/nonwhite}

## Preguntas Frecuentes

***¿Dónde puedo ver/editar mis Plantillas guardadas?***

Los Diseños que se han guardado como Plantillas se ven y gestionan desde la página de Plantillas bajo la sección Diseño del menú principal del CMS.

{nonwhite}

***¿No veo el botón Xibo Exchange desde la Caja de Herramientas?***

{cloud}
Estas Plantillas están habilitadas por defecto para nuevos clientes de Xibo Cloud Hosting. Si estás actualizando desde una versión anterior necesitarás habilitar el **Conector Xibo Exchange** que se encuentra bajo la sección **Administrador** del menú principal del CMS.
{/cloud}

{noncloud}
Las instalaciones locales no tienen estas Plantillas habilitadas por defecto. Habilita el **Conector Xibo Exchange** que se encuentra bajo la sección **Administrador** del menú principal del CMS.
{/noncloud}
{/nonwhite}
