# Visión General del Editor de Diseños

El Editor de Diseños se utiliza para crear nuevos Diseños y cuando un Diseño existente necesita un cambio.

Un Diseño presenta contenido para ser mostrado en Pantallas y se compone de solo 1 o múltiples elementos de contenido. Los diseños pueden incluir una combinación de **Elementos**, **Widgets** y **Multimedia de Librería** y pueden añadirse para mostrarse en diferentes secciones del Diseño simultáneamente.

El Editor de Diseños de [[PRODUCTNAME]] ofrece a los usuarios una gama de herramientas para añadir y organizar una variedad de contenido, independientemente del nivel de habilidad, para crear fácilmente diseños llamativos que capten la atención en las Pantallas.

## Creando Diseños

Los Diseños se crean desde la página **Diseños** bajo la sección **Diseño** del menú principal del CMS.

Haciendo clic en el botón **Añadir Diseño** se abrirá el **Editor de Diseños**, usado para añadir contenido y diseñar.

## Navegando por el Editor de Diseños

![Visión General del Editor de Diseños](img/layout_editor_overview_explainer.png)

Para **Renombrar** el Diseño, haga clic en el texto "Sin Título" en la parte superior del Editor de Diseños.

Desde aquí, también puede ver el **Estado** y la **Duración** total del Diseño mientras construye su diseño. La duración total del Diseño se basa siempre en el elemento añadido al Diseño con la mayor duración establecida.

Los Diseños funcionarán mejor cuando se muestren en una Pantalla con una resolución coincidente. Para cambiar las **Dimensiones** de un Diseño, seleccione la **Resolución** a usar desde el menú desplegable en el **Panel de Propiedades**.

Usando el **Panel de Propiedades**, seleccione un **Color** o añada una **Imagen** estática desde la **Caja de Herramientas** o suba un nuevo archivo para usar como **Fondo** para el Diseño.

Todo el contenido se añade al lienzo desde la **Caja de Herramientas**.

### Añadiendo y Organizando Contenido

La **Caja de Herramientas** contiene una variedad de tipos de contenido que, al seleccionarse, abrirán los elementos para añadir.

El contenido se añade al lienzo usando la flexibilidad de arrastrar y soltar, junto con herramientas de rotación y redimensionamiento para permitir a los Usuarios posicionar los elementos exactamente donde se necesitan. 

Al añadir un elemento al Lienzo, los botones de "Ajustar a" están disponibles en la parte inferior derecha bajo el lienzo, lo que permite un posicionamiento más fácil al mover elementos por el lienzo.

### Panel de Propiedades

A medida que se añaden elementos, el **Panel de Propiedades** mostrará las opciones de configuración disponibles para el elemento seleccionado.

La pestaña **Avanzado** se puede usar para anular la duración de un elemento.

- Haga clic en **Establecer una duración** y proporcione la nueva **Duración** en segundos.

Si este elemento tiene ahora la duración más larga en el Diseño, también actualizará la duración total del Diseño mostrada en la parte superior del Editor de Diseños.

### Previsualización del Diseño

Se puede reproducir el Diseño completo para previsualizar cómo se verá su diseño al mostrarse en las Pantallas.

- Deseleccione cualquier elemento haciendo clic en el fondo.
- Seleccione el icono del botón **Reproducir** en la parte inferior izquierda bajo el lienzo.

{tip}
¡Reproduzca una vista previa a pantalla completa usando el icono en la parte superior derecha del lienzo antes de seleccionar el botón Reproducir!
{/tip}

### Publicando Diseños

Los Diseños necesitan ser **Publicados** para mostrarse correctamente en las Pantallas cuando se incluyen en Programaciones. 

Use el menú de 3 líneas situado en la parte superior derecha del Editor de Diseños para seleccionar la opción **Publicar**.

Los Diseños pueden Publicarse inmediatamente o se puede establecer una **Fecha de Publicación**, desmarcando la opción **Publicar Ahora** para determinar una fecha y hora específicas en el futuro para Publicar.

Una vez Publicado o tras establecer una Fecha de Publicación, el Diseño ya puede ser Programado en las Pantallas.

## Preguntas Frecuentes (FAQs)

***¿Cuándo debería un Usuario crear un nuevo Diseño?***

El uso recomendado para un Diseño es que contenga solo el diseño y los medios necesarios para un conjunto de contenido relacionado que deba mostrarse al mismo tiempo. ¡Mantener su contenido conciso hará que sea más fácil para su audiencia entender su mensaje!

***¿Hay restricciones en la cantidad de Diseños que puedo tener?***

No hay límites para el número de Diseños en el sistema o el número que un Usuario puede tener.

***El estado de mi Diseño siempre muestra un signo de exclamación naranja, ¿cómo hago para que sea un tic verde?***

El signo de exclamación naranja es un estado esperado si su Diseño contiene contenido online, como una Página Web. En esta situación, el CMS no puede determinar qué reproducirá la Pantalla ya que este tipo de contenido no se envía directamente desde el CMS y se evalúa en el lado del Reproductor. El Diseño aún debe ser Publicado y Programado en las Pantallas, las cuales necesitarán tener una conexión a internet fiable para mostrar este contenido correctamente.

***Quiero dividir mi Diseño y tener contenido estático a un lado y un carrusel de contenido llenando el otro lado, ¿cómo lo consigo?***

Añada y posicione una Lista de Reproducción en el lado donde quiere mostrar una rotación de elementos y redimensione para ajustar al área según sea necesario.

***¿Cómo hago cambios de diseño en un Diseño existente?***

Use el menú de fila para el Diseño en el que necesita hacer cambios y seleccione Diseño para abrir el Editor de Diseños. Si el Diseño ha sido Publicado, necesitará primero Extraer (Checkout) el Diseño, lo cual creará una copia borrador del Diseño para no interrumpir ninguna programación actual en la que esté el Diseño.
