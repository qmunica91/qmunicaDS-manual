# Vídeo

Añade archivos de Vídeo para reproducir en tus Diseños y Listas de Reproducción desde la [Búsqueda en la Biblioteca](layouts_editor_using_library_search).

{feat}Video|v4{/feat}

## Descripción General

- [Sube](media_library.html#content-add-media-upload) nuevos archivos de vídeo directamente a Listas de Reproducción/Diseños desde la Búsqueda en la Biblioteca.
- Los archivos subidos directamente a Listas de Reproducción y Diseños se guardan automáticamente en la [Biblioteca](media_library.html) para su reutilización.
- Los archivos de vídeo también pueden subirse por adelantado a la Biblioteca.

{tip}
Recomendamos formato de archivo H264 MP4.

Al subir archivos de vídeo, la barra de desplazamiento se puede usar para seleccionar un cuadro para ser usado como miniatura en el CMS para una identificación más fácil.
{/tip}

- Establece [tiempos de inicio y fin](media_playlists.html#content-widget-expiry-dates) para archivos de vídeo subidos directamente a una Lista de Reproducción.

- Guarda archivos de vídeo en Carpetas al subir, para controlar fácilmente el acceso a los Usuarios.

- Los archivos de vídeo subidos se almacenan en caché para reproducción fuera de línea.

- Controla la duración del archivo de vídeo.
  - Establece una duración menor para cortar el archivo original.
  - Establece una duración mayor y usa la función de Bucle para "volver" al principio para satisfacer una duración más larga.

- Opciones de escalado de Aspecto y Estirar.

- Expande la parte superior del contenido existente para mostrar el vídeo a pantalla completa.

- Silencia vídeos que tienen audio.

- Selecciona vídeos para usar desde Pixabay sin salir del CMS.

{version}
**NOTA:**
El escalado es actualmente compatible con Reproductores Android, webOS y Tizen (desde v2 R200) y Reproductores Windows (desde v2 R253).
Las transiciones no son compatibles para el Reproductor Tizen cuando se usa el Widget de Vídeo.
{/version}
