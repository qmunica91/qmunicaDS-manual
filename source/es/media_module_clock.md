# Reloj

Selecciona entre un estilo de reloj **Analógico**, **Flip** o **Digital**.

{feat}Clock|v4{/feat}

## Descripción General

- Muestra una cara de Reloj Analógico tradicional en un tema claro u oscuro.
- Especifica el formato de hora a usar para tu Reloj Digital ingresando texto / HTML o usa los Formatos de Fecha suministrados.
- Muestra un Reloj Flip animado e incluye mostrar los segundos transcurridos.
- Usa las opciones de Contador Horario, Minuto y Diario con el Reloj Flip y ejecuta desde una fecha/hora especificada para mostrar una cuenta regresiva.
- Aplica un desplazamiento en minutos a la hora actual.
