# Notificaciones

Muestra Notificaciones en Diseños y Listas de Reproducción, así como mensajería dirigida a Pantallas/Grupos de Pantallas usando el Widget de Notificación.

Los usuarios pueden crear mensajes haciendo clic en **Centro de Notificaciones** al que se accede haciendo clic en el icono de campana de alarma junto al Perfil de Usuario.

{tip}
¡Los Diseños ya deben estar en la programación y contener el Widget de Notificación para dirigir a Pantallas/Grupos de Pantallas con éxito!
{/tip}

## Elementos de Notificación

Los Elementos están disponibles para selección al añadir el Widget de Notificación a un [Diseño](layouts_editor.html) para dar a los Usuarios más control sobre qué componentes de las Notificaciones usar y dónde se pueden colocar.

![Notification Elements](img/v4_media_module_notification_elements.png)

Cada Elemento tiene un conjunto de opciones de configuración en el Panel de Propiedades. Controla cómo los artículos deben ciclarse especificando una [Ranura de Datos](layouts_editor.html#content-data-slots) para usar para cada uno de los Elementos añadidos. Los Elementos de Datos pueden complementarse aún más añadiendo Elementos Globales para añadir formas y texto que se pueden poner en un Grupo de Elementos para una configuración y posicionamiento más fáciles.

## Descripción General

- Actualiza Elementos con nuevas Notificaciones creándolas en el Cajón de Notificaciones.
- Actualiza contenido sin acceder a Diseños.
- Mensajería dirigida a Pantallas/Grupos de Pantallas.
- Especifica una antigüedad máxima en la que mostrar mensajes.
- Establece una duración para aplicar a cada mensaje individual.
- Aplica un número de elementos para mostrar.
- El contenido para este medio es almacenado en caché por los Reproductores para reproducción fuera de línea.
