# Tableros de Menú: Productos

Muestra datos para Productos contenidos en un Tablero de Menú en cualquier lugar de un Diseño usando **Elementos**.

{feat}Menu Board: Products|v4{/feat}

El Widget de Tableros de Menú: Productos consiste principalmente en una fuente de Tablero de Menú que alimenta los Elementos configurados.

{tip}
Los [Tableros de Menú](media_menuboards.html) deben crearse con Productos añadidos antes de añadir el Widget de Tableros de Menú: Productos a los Diseños.
{/tip}

## Elementos de Producto

Los Elementos están disponibles para selección al añadir un Widget de Tableros de Menú: Producto a un [Diseño](layouts_editor.html) para dar a los Usuarios más control sobre qué componentes de los Productos de Tablero de Menú usar y dónde se pueden colocar.

![Product Elements](img/v4_media_modules_product_elements.png)

Cada Elemento tiene un conjunto de opciones de configuración en el Panel de Propiedades. Se debe seleccionar un Tablero de Menú para usar como fuente de datos desde la pestaña **Configurar** para cada Elemento usado en el Diseño. Controla cómo los artículos deben ciclarse especificando una [Ranura de Datos](layouts_editor.html#content-data-slots) para usar para cada uno de los Elementos añadidos. Los Elementos de Datos pueden complementarse aún más añadiendo Elementos Globales para añadir formas y texto que se pueden poner en un Grupo de Elementos para una configuración y posicionamiento más fáciles.

## Descripción General

- Actualiza Elementos con nuevos datos de Producto editando los datos subyacentes del [Tablero de Menú](media_menuboards.html#content-create-and-configure-categories).
- Actualiza el contenido del Widget de Tableros de Menú: Productos sin acceder a los Diseños.
- Selecciona cómo deben ordenarse los elementos.
- Elige mostrar Productos que han sido marcados como no disponibles.
- Los elementos no disponibles se pueden mostrar 'atenuados' para ser menos prominentes.
- La duración se puede establecer por elemento.
- El contenido para este medio es almacenado en caché por los Reproductores para reproducción fuera de línea.
- Úsalo con el Widget [Tableros de Menú: Categoría](media_module_menuboards_category.html) para incluir información de encabezado para complementar la presentación de datos de Producto para mostrar en Pantallas.
