# Gestión de Conectores

Los Módulos que dependen de un servicio de terceros para datos, [Acciones](media_module_stocks.html) por ejemplo, necesitan una configuración adicional con claves API, etc.

Una vez que el Módulo ha sido habilitado y tienes la clave API requerida:

- Ve a la página **Aplicaciones** bajo la sección **Administración** del menú principal del CMS.
- Desplázate hacia abajo hasta la sección **Conectores** de la página.

Aquí puedes ver todos los servicios de terceros que se pueden configurar:

![Connectors](img/v4_media_modules_connectors.png)

- Haz clic en el botón **Configurar** del Conector para configurar:

![Configure Connectors](img/v4_media_modules_configure_connectors.png)

- Proporciona la **clave API** que se te ha dado y asegúrate de marcar la casilla **Habilitado** para decirle al Conector que comience a proporcionar el servicio a los Módulos relevantes.
