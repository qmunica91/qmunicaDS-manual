## Selector Rectangular

Se pueden mover varios elementos juntos para mantener la alineación en un bloque usando la herramienta **Selector Rectangular**:

- Haz clic en el fondo del lienzo.
- Mantén presionado y arrastra sobre el área que deseas seleccionar para crear un contorno. 
- Asegúrate de que todos los elementos que se moverán juntos estén contenidos.

- Arrastra todos los elementos para posicionar en el lienzo.
- Una vez en posición, haz clic fuera del selector rectangular para salir.

## Agrupando Elementos

Facilita la construcción de diseños en el Editor de Diseños **Agrupando Elementos**:

- Añade Elementos al lienzo.

Al agrupar Elementos de Datos, deben ser del mismo tipo de Widget de Datos (todos Elementos de Clima por ejemplo) y compartir el mismo Espacio de Datos y Efecto.

- Posiciona y estiliza cada Elemento usando la pestaña **Apariencia**.
- Manteniendo presionada la tecla shift, haz clic en cada **Elemento** para seleccionar para agrupar.
- Una vez seleccionados, suelta la tecla shift y haz clic derecho.
- Desde el menú, selecciona **Agrupar elementos**.

El grupo ahora se puede mover alrededor del lienzo para reposicionar y redimensionar.

{tip}
¡Los Elementos Agrupados se pueden duplicar fácilmente usando el Menú Contextual!
{/tip}

- Para hacer más ediciones a Elementos individuales mientras están agrupados, haz clic en el icono **Editar Grupo** en la esquina superior derecha del grupo.
- Haz clic en un Elemento para hacer cambios en el Panel de Propiedades.
- Selecciona **X** para salir de la edición del grupo.

Para **Desagrupar**, usa el Menú Contextual.

## Gestor de Capas

Ve y gestiona la capa y duraciones de todos los elementos añadidos al Diseño usando el **Gestor de Capas** ubicado en la parte inferior izquierda del Editor de Diseños:

- Usa esta lista para arrastrar y soltar el orden para mostrar elementos delante o detrás de otro contenido.
- Alternativamente haz clic derecho en un elemento seleccionado para usar las opciones **Enviar/Traer** desde el **Menú Contextual**.

Los Elementos tienen su propia Capa de Lienzo que se puede usar para determinar dónde aparecen en relación con otros Elementos renderizados nativamente como Listas de Reproducción y Vídeos.

## Menú Contextual

Haz clic derecho en elementos que se han añadido a un Diseño o Lista de Reproducción para seleccionar de un menú de opciones adicionales (relevantes para el elemento seleccionado).

Al hacer clic en un **Widget**, hay más opciones disponibles, mostradas bajo el nombre/tipo del Widget.

## Menú de Opciones

Úsalo para mostrar/ocultar **Tooltips** así como proporcionar una advertencia de **Confirmación de eliminación** para todos los elementos añadidos al Diseño.

Desde aquí selecciona **Limpiar Lienzo** para comenzar tu diseño de nuevo o **Nuevo** para crear un Diseño completamente nuevo.

## Preguntas Frecuentes

***¿Por qué no puedo agrupar un Elemento de Moneda con un Elemento de Tablero de Menú?***

Los Elementos de Datos deben ser del mismo tipo de Widget de Datos para poder agruparlos. ¡También deben compartir el mismo Espacio de Datos y cualquier Efecto establecido!

***¿Se pueden agrupar los Elementos Globales con Elementos de Datos?***

Sí, incluir Elementos Globales en grupos es una excelente manera de complementar fácilmente los Elementos de Datos añadiendo texto, formas y líneas.
