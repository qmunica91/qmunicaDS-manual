# Ordenar y Filtrar

La información de la página dentro del CMS se presenta como datos tabulares en formato de cuadrícula. Las **Cuadrículas** tienen una serie de características combinadas crean una herramienta flexible para los Usuarios, facilitando la visualización y comprensión de la información presentada.

Se pueden usar múltiples campos de filtro para restringir los criterios de los resultados devueltos.

- Usa la casilla de verificación para aplicar opciones de filtrado avanzadas para Nombres usando Regex y proporcionar criterios Y/O usando términos separados por comas.
- La pestaña **Avanzado** proporciona acceso a más opciones de filtrado para las páginas.

Los resultados se completan en columnas, que pueden ordenarse haciendo clic en el encabezado de la columna o usando la tecla Mayús para ordenar múltiples columnas que contienen flechas arriba/abajo.

- Mantén presionada la tecla Mayús y haz clic en los encabezados de columna para ordenar en el orden seleccionado.

El botón **Visibilidad de Columna** se usa para seleccionar/deseleccionar qué columnas son necesarias en la tabla de datos, convirtiéndolo en una herramienta extremadamente útil para que los Usuarios personalicen qué columnas deben ser visibles para cada cuadrícula.

Si el número de columnas seleccionadas no cabe en el tamaño de tu pantalla, la información se ocultará como filas adicionales, indicadas por un icono + al principio de la fila. Al hacer clic en este icono se mostrará la información oculta.

En la parte inferior de las cuadrículas seleccionadas, está disponible una opción **Con Seleccionados** para que los usuarios seleccionen múltiples filas para realizar fácilmente acciones masivas, como Eliminar, Compartir y mover elementos a Carpetas.

Usa el menú de fila para acceder a un conjunto de acciones para un elemento seleccionado. Las acciones del menú de fila seleccionadas a través de las cuadrículas tienen una capacidad de 'envío automático'. Una vez marcada, la acción se enviará automáticamente sin más requisitos de confirmación cuando se seleccione la próxima vez.
