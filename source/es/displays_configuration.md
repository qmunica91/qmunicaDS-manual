# Configuración de Pantallas

Una vez que las Pantallas se han conectado, se configuran y gestionan desde la página de **Pantallas** dentro del CMS.

{nonwhite}

## Vídeo Explicativo

{video}DsYGiDgiigM|how_to_display_configuration.png{/video}
{/nonwhite}

Para comenzar a mostrar contenido, las **Pantallas** deben estar **Autorizadas** con el CMS. Comprueba la cuadrícula de Pantallas para asegurarte de que las Pantallas correctas se muestran como **Autorizadas**, como primer paso.

- Dale a la **Pantalla** un nombre amigable para facilitar su identificación dentro del CMS utilizando la opción **Editar** del menú de fila.

Ahora es un buen momento para asignar a tu Pantalla un **Diseño por Defecto**. 

- Desplázate hacia abajo en el formulario para seleccionar un Diseño para usar como predeterminado. Este Diseño se mostrará cuando no haya contenido programado o surja un problema que impida mostrar el contenido programado en tu Pantalla.

{tip}
¡Al crear tus propios Diseños por Defecto, ten en cuenta que los diseños deben mantenerse simples, evitando cualquier contenido multimedia y web complejo!
{/tip}

- Haz clic en la pestaña **Detalles** para proporcionar más información sobre tu Pantalla.

- Establecer la **Latitud** y **Longitud** de la Pantalla mostrará su ubicación en la vista de mapa.

El mapa de **Pantallas** se puede ver haciendo clic en el icono junto al interruptor de **Carpetas** desde la cuadrícula de Pantallas:

![Display Map Icon](img/display_configuration_map_view_icon.png)

- Establece **Zonas Horarias** para asegurar que el contenido se muestre en el momento correcto para la ubicación de tu Pantalla.

A cada Pantalla se le asigna un perfil predeterminado que aplicará automáticamente sus configuraciones a todas las Pantallas del mismo tipo. Por ejemplo, editas el Intervalo de Recolección para tu Perfil de Pantalla Android. Una vez guardado, el nuevo tiempo de recolección se aplicará a todas las Pantallas asignadas con este Perfil de Pantalla.

Los **Perfiles de Configuración de Pantalla** se pueden personalizar para la gestión automática y configuración de tu Red de Pantallas.

## Preguntas Frecuentes

***¿Qué sucede si no tengo contenido programado y mi diseño por defecto tiene un problema para descargar su contenido?***

Se mostrará la pantalla de bienvenida hasta que se resuelva el problema con el diseño por defecto o se programe contenido para mostrar en la Pantalla.

***¿Qué indica el Icono de Nube en la columna de Estado de la Cuadrícula de Pantallas?***

El Icono de Nube muestra que el Reproductor no ha iniciado sesión y el contenido está esperando ser descargado. Una Cruz indica que el Reproductor aún no ha completado la descarga y un Tic muestra que el Reproductor está actualizado.
