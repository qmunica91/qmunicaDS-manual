# Cambios de Diseño de Layout

De vez en cuando, los **Diseños** pueden necesitar cambios/actualizaciones de diseño para mantener el contenido fresco y relevante. [[PRODUCTNAME]] continuará mostrando el diseño de Layout Publicado ya en las Programaciones de Pantalla permitiendo a los Usuarios realizar cambios importantes en una versión borrador verificada del Diseño. Solo los cambios aprobados sobrescribirán el diseño existente una vez que un Usuario seleccione Publicar.

{nonwhite}

## Vídeo Explicativo

{video}5NdKMd8QAk0|how_to_design_changes.png{/video}
{/nonwhite}

Los Diseños que están en estado **Borrador** se editan simplemente usando el menú de fila y seleccionando **Diseño** para abrir el Editor de Diseños.

Seleccionar **Diseño** para un **Diseño Publicado** mostrará que está bloqueado para edición.

- Haz clic en **Ver** para ver todos los componentes del Diseño en modo de solo lectura.
- Haz clic en **Checkout** para crear un borrador del Diseño para habilitar la edición.

Una vez que se ha realizado un "checkout" de un Diseño, se hace una copia del Diseño en estado borrador. Este Diseño borrador no reflejará ningún cambio en la versión publicada. Cualquier programación de **Pantalla** continuará mostrando la versión **Publicada** de este **Diseño** permitiendo a los Usuarios tomarse su tiempo con cualquier cambio de diseño que deba realizarse.

{tip}
Los Diseños también pueden pasar por "checkout" desde el menú de fila antes de que un Usuario necesite realizar cambios de diseño. ¡Selecciona la casilla de verificación 'envío automático' para enviar el formulario automáticamente sin la ventana de aprobación adicional cada vez que se selecciona Checkout!
{/tip}

## Bloqueo de Layout

Para evitar que múltiples Usuarios realicen ediciones en el mismo Diseño al mismo tiempo, el Diseño se bloqueará.

El Diseño permanecerá bloqueado para otros Usuarios durante el tiempo especificado en el mensaje de bloqueo. El Diseño puede ser desbloqueado por el Usuario actual usando el menú Opciones y seleccionando **Desbloquear**. Una vez desbloqueado, el usuario actual será redirigido de vuelta a la cuadrícula de Diseños.

Para revertir el Diseño a su estado original publicado, usa el menú **Opciones** y selecciona **Descartar Borrador**.

## Opciones de Publicación

Para hacer permanentes los cambios y sobrescribir el Diseño Publicado existente, selecciona **Publicar**.

Los Diseños pueden publicarse inmediatamente usando la casilla de verificación **Publicar Ahora**, o desmarcando esta opción se permite al Usuario establecer una fecha y hora como **Fecha de Publicación**.

{tip}
¡Establecer una **Fecha de Publicación** no es una alternativa a crear un evento en una programación!
{/tip}

Una vez publicado, el CMS actualizará un cambio de Diseño para que los cambios de diseño aprobados se muestren en las Pantallas.

## Preguntas Frecuentes

***¿Puedo eliminar el envío automático de formularios de Checkout?***

Los formularios que se han habilitado para 'envío automático' se pueden ver y gestionar desde la configuración del Perfil de Usuario.

***¿Puedo configurar Diseños para que se publiquen automáticamente?***

Hay una configuración predeterminada que se puede establecer para Publicar automáticamente todos los Diseños Borrador 30 minutos después de la última edición del Usuario. Esto se encuentra en la página de Configuración bajo la sección Administración del menú principal del CMS.
