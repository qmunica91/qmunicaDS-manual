# Mostrar un Vídeo en Pantallas

Añade un archivo de **Vídeo** a tu **Biblioteca Multimedia** y **Prográmalo** para reproducirse en **Pantallas**.

Es posible que tengas contenido de vídeo creado para mostrar promociones de productos o actualizaciones de la empresa a los empleados, listo para ser mostrado a pantalla completa en tu red de pantallas.

Los archivos de vídeo se añaden primero a la Biblioteca mediante subida o URL. Para este ejemplo, se subirá un archivo de vídeo a la Biblioteca Multimedia del CMS.

### Añadir Multimedia

- Haz clic en **Multimedia** bajo la sección **Biblioteca** del menú principal del CMS.
- Haz clic en el botón **Añadir Multimedia**.
- Selecciona **Añadir archivos** y elige el archivo de vídeo a subir.
- Usa la barra de desplazamiento para seleccionar un fotograma para usar como miniatura del vídeo, y así facilitar su identificación en las listas de la Biblioteca Multimedia.
- Proporciona un nuevo **Nombre** o deja este campo en blanco para mantener el nombre original del archivo.
- Haz clic en **Iniciar subida**.

- Una vez subido, haz clic en **Hecho**.

### Seleccionar Multimedia

- Desde la cuadrícula de **Multimedia**, usa los filtros para facilitar la búsqueda del archivo de vídeo subido si es necesario.
- Usa el menú de fila y selecciona **Programar**:

Se abrirá el formulario **Seleccionar Multimedia** para proporcionar opciones de configuración adicionales para el Vídeo:

**Duración en bucle** determina cuánto tiempo se reproducirá el vídeo cada vez que aparezca en un **Bucle de Programación**. 

- Establece una duración menor para cortar el archivo original o una duración mayor para que el vídeo se "repita" desde el principio hasta completar el tiempo establecido.
- Selecciona una **Resolución** específica para usar.
- Establece un **color** para rellenar los huecos en el fondo si el archivo multimedia no llena la pantalla.
- **Guardar** para cerrar el formulario **Seleccionar Multimedia** y continuar con la programación.

### Programar Evento

 Completa los campos en el formulario **Programar Evento**:

- Dale al evento un **Nombre** para facilitar su identificación en el CMS.
- Selecciona en qué **Pantallas** se mostrará este Vídeo.
- Desde **Partición del día**, selecciona **Siempre** para mostrar el Vídeo constantemente o selecciona **Personalizado** para establecer fechas/horas de inicio y fin específicas.
- Haz clic en **Guardar.**

El CMS actualizará ahora un cambio de Programación y tu Vídeo se mostrará en las Pantallas durante los tiempos establecidos en la programación.

## Preguntas Frecuentes

***¿Qué son los Bucles de Programación?*** 

Un Bucle de Programación es cómo se reproducirá el contenido multimedia cuando esté programado con otros contenidos multimedia. Por ejemplo, si has programado 2 elementos multimedia cada uno con una duración de 10 segundos para mostrarse al mismo tiempo durante 1 hora, ambos elementos se reproducirán durante sus duraciones de 10 segundos en rotación equitativa durante el tiempo programado.

***¿Qué formato de archivo se recomienda para Vídeos?***

Recomendamos el formato de archivo H264 MP4.
