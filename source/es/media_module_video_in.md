# Entrada de Vídeo (Video In)

Úsalo para mostrar entrada de una fuente externa en Pantallas SoC.

{feat}Video In|v4{/feat}

{version}
**NOTA:** Este Widget es compatible para uso con Reproductores webOS y Tizen, así como Reproductores System on Chip de Philips que ejecutan Android 8 solamente.
{/version}

### Descripción General

- Selecciona de múltiples entradas de dispositivo.
  - HDMI (soporte para HDMI 1 solamente)
  - RGB
  - DVI
  - DP
  - OPS
- Expande el contenido sobre el contenido existente para mostrar el vídeo a pantalla completa.

{version}
**NOTA:** Las transiciones no son compatibles para el Reproductor Tizen cuando se usa este Widget.
{/version}
