# Vídeo Local

Añade un vídeo desde una ruta de archivo local o URL a Listas de Reproducción y Diseños. Este tipo de vídeo solo existe en el Reproductor y no se sube al CMS.

{feat}Local Video|v4{/feat}

## Descripción General

- Muestra vídeos transferidos manualmente.
- Úsalo con vídeos en una LAN.
- Transmite vídeos.
- La ruta del archivo o URL puede ser a una transmisión RTSP.
- Opciones de escalado de Aspecto y Estirar.
- Funcionalidad de silencio (Mute).
- Expande el contenido sobre el contenido existente para mostrar el vídeo a pantalla completa.

{version}
**NOTA:** El escalado de vídeo y las transmisiones RTSP solo son compatibles con Reproductores Android, webOS y Linux. Usa el Widget [HLS](media_module_hls.html) para mostrar transmisiones de vídeo compatibles en Reproductores Windows y Tizen.
El número de transmisiones RTSP que puedes mostrar en un Diseño / Lista de Reproducción depende del dispositivo que se esté utilizando.
Las transiciones no son compatibles para el Reproductor Tizen cuando se usa el Widget de Vídeo Local.
{/version}
