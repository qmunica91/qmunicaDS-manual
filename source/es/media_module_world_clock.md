# Reloj Mundial

Selecciona entre relojes de estilo Analógico, Hora y Fecha o Texto para mostrar Zonas Horarias seleccionadas.

{feat}World Clock|v4{/feat}

## Descripción General

- Muestra múltiples Zonas Horarias en tu estilo de Reloj Mundial elegido.
- Establece el número de Columnas y Filas cuando se han seleccionado múltiples Zonas Horarias.
- Incluye tu propia Etiqueta para cada Zona Horaria seleccionada.
- Opcionalmente selecciona para Resaltar.

- Configuraciones de apariencia personalizables.
- Configuraciones personalizadas adicionales para el Reloj Analógico para mostrar aguja de segundos, mostrar pasos, reloj digital interior por ejemplo.
