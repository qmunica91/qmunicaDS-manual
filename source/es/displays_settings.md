# Ajustes de Pantalla 

Cada Pantalla tiene configuraciones que pueden configurarse individualmente. Los ajustes pueden variar según la plataforma/tipo de reproductor multimedia, mediante el uso de **Perfiles de Configuración de Pantalla** puedes aplicar fácilmente diferentes configuraciones a **Pantallas** de la misma plataforma o tipo.

Una vez conectada al CMS, a cada **Pantalla** se le asigna un **Perfil de Pantalla** por defecto para aplicar configuraciones a las Pantallas del tipo correspondiente.

Los perfiles se pueden personalizar o añadir nuevos para adaptarse a los requisitos de la Pantalla. Para mayor facilidad, utiliza el menú de fila y **Copia** un **Perfil** existente para crear rápidamente nuevos Perfiles con la configuración necesaria.

Usa Perfiles de Pantalla para proporcionar tu **Código de Grupo de Licencias** a todas tus Pantallas de forma centralizada para un licenciamiento más fácil y menos propenso a errores que ingresar códigos manualmente por dispositivo.

Crea Perfiles de Pantalla para atender a las **orientaciones** Horizontal y Vertical o para aplicar diferentes **horas de funcionamiento** o un Perfil de Pantalla para dispositivos que requieren **múltiples decodificadores** habilitados, por ejemplo.

Si Pantallas individuales requieren configuraciones diferentes a las establecidas en un Perfil de Pantalla, anula la configuración editando la Pantalla desde la cuadrícula de Pantallas.

## Preguntas Frecuentes

***¿Tengo una Pantalla que debería usar un Perfil alternativo al establecido por defecto?***

Anula Pantallas individuales desde la cuadrícula de Pantallas. Haz clic para Editar y usa la pestaña Configuración para seleccionar un Perfil de Configuración alternativo para usar en la Pantalla seleccionada.
