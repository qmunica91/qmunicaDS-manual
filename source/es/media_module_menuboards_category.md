# Tableros de Menú: Categoría

Muestra datos para Categorías contenidas en un Tablero de Menú en cualquier lugar de un Diseño usando **Elementos**.

{feat}Menu Board: Category|v4{/feat}

El Widget de Tableros de Menú: Categoría consiste principalmente en una fuente de Tablero de Menú que alimenta los Elementos configurados.

{tip}
Los [Tableros de Menú](media_menuboards.html) deben crearse con Categorías definidas antes de añadir el Widget de Tableros de Menú: Categoría a los Diseños.
{/tip}

## Elementos de Categoría

Los Elementos están disponibles para selección al añadir un Widget de Tableros de Menú: Categoría a un [Diseño](layouts_editor.html) para dar a los Usuarios más control sobre qué componentes de las Categorías de Tablero de Menú usar y dónde se pueden colocar.

![Category Elements](img/v4_media_modules_category_elements.png)

Cada Elemento tiene un conjunto de opciones de configuración en el Panel de Propiedades. Se debe seleccionar un Tablero de Menú para usar como fuente de datos desde la pestaña **Configurar** para cada Elemento usado en el Diseño. Controla cómo los artículos deben ciclarse especificando una [Ranura de Datos](layouts_editor.html#content-data-slots) para usar para cada uno de los Elementos añadidos. Los Elementos de Datos pueden complementarse aún más añadiendo Elementos Globales para añadir formas y texto que se pueden poner en un Grupo de Elementos para una configuración y posicionamiento más fáciles.

## Descripción General

- Actualiza Elementos con nuevos datos de Categoría editando los datos subyacentes del [Tablero de Menú](media_menuboards.html#content-create-and-configure-categories).
- Actualiza el contenido del Widget de Tableros de Menú: Categoría sin acceder a los Diseños.
- El contenido para este medio es almacenado en caché por los Reproductores para reproducción fuera de línea.
- Úsalo con el Widget [Tableros de Menú: Productos](media_module_menuboards_products.html) para crear fácilmente un menú/catálogo de artículos para que tu Categoría se muestre en Pantallas.
