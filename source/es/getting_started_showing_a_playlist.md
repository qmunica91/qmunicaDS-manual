# Mostrar una Lista de Reproducción en Pantallas

**Crea** una **Lista de Reproducción** de contenido y **Prográmala** para mostrar en **Pantallas**.

Es posible que tengas un conjunto de contenido recopilado para satisfacer un requisito específico que necesita mostrarse en una secuencia rotativa en las Pantallas. Las listas de reproducción permiten a los usuarios agilizar recursos y organizar múltiples tipos de contenido para reproducirse en el orden que necesitan ser mostrados.

Las listas de reproducción pueden tener contenido añadido mediante asignación automática basada en criterios seleccionados o mediante selección manual. Para este ejemplo, el contenido de imagen ya subido a la Biblioteca Multimedia del CMS se añadirá a una Lista de Reproducción manualmente.

### Añadir Lista de Reproducción

- Haz clic en **Listas de Reproducción** bajo la sección **Biblioteca** del menú principal del CMS.
- Haz clic en el botón **Añadir Lista de Reproducción**.
- Proporciona un **Nombre** para facilitar su identificación en el CMS.
- Haz clic en Guardar y se abrirá el **Editor de Listas de Reproducción**.

### Añadir Contenido

- Usando la Caja de Herramientas, haz clic en **Búsqueda de imagen de biblioteca** para añadir Imágenes de la Biblioteca del CMS.
- Usa los filtros para localizar las **Imágenes** a añadir.
- Arrastra y suelta imágenes para añadirlas a la **Línea de Tiempo**.
- Reordena la secuencia arrastrando y soltando.

Puedes ver la duración total de la Lista de Reproducción en la parte superior de la Línea de Tiempo.

- Para cambiar la duración de los elementos en la Lista de Reproducción, haz clic para resaltar y selecciona la pestaña **Avanzado** en el **Panel de Propiedades**.
- Marca la casilla **Establecer una duración** y proporciona una nueva duración en segundos.
- Haz clic en el botón **Guardar** para actualizar la duración total de la Línea de Tiempo.
- Selecciona **Salir** para volver a la cuadrícula de Listas de Reproducción.

### Seleccionar Lista de Reproducción

- Desde la cuadrícula de Listas de Reproducción usa los filtros para localizar la Lista de Reproducción.
- Usa el menú de fila y selecciona **Programar**.

Se abrirá el formulario **Seleccionar Lista de Reproducción** para proporcionar opciones de configuración adicionales para la Lista de Reproducción:

- Selecciona una **Resolución** específica para usar.
- Establece un **color** para rellenar los huecos en el fondo si la Lista de Reproducción no llena la pantalla.
- **Guardar** para cerrar el formulario **Seleccionar Lista de Reproducción** y continuar con la programación.

### Programar Evento

 Completa los campos en el formulario **Programar Evento**:

- Dale al evento un **Nombre** para facilitar su identificación en el CMS.
- Selecciona en qué **Pantallas** se mostrará esta Lista de Reproducción.
- Desde **Partición del día**, selecciona **Personalizado** para establecer fechas/horas de inicio y fin específicas o selecciona **Siempre** para mostrar la Lista de Reproducción constantemente.
- Haz clic en **Guardar.**

El CMS actualizará ahora un cambio de Programación y la Lista de Reproducción se mostrará en las Pantallas durante los tiempos establecidos en la programación.

## Preguntas Frecuentes

***¿Puedo añadir Widgets a una Lista de Reproducción?***

Sí, se pueden añadir Widgets usando el Editor de Listas de Reproducción utilizando el botón superior de la Caja de Herramientas.

***¿No veo Elementos Globales o de Datos para añadir a mi Lista de Reproducción?***

Los Elementos no están disponibles en el Editor de Listas de Reproducción. Los Widgets de Datos tienen una selección de Plantillas Estáticas para elegir cómo mostrar tus datos. 

***¿Tengo que subir los archivos que quiero usar a la Biblioteca Multimedia del CMS antes de crear una Lista de Reproducción?***

No, puedes subir multimedia directamente a una Lista de Reproducción arrastrando la tarjeta de Subida para un tipo de medio seleccionado a la Línea de Tiempo y usar el cargador de archivos.
