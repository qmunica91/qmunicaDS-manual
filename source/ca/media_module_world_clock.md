# Rellotge Mundial

Selecciona entre rellotges d'estil Analògic, Hora i Data o Text per mostrar Zones Horàries seleccionades.

{feat}World Clock|v4{/feat}

## Descripció General

- Mostra múltiples Zones Horàries en el teu estil de Rellotge Mundial triat.
- Estableix el nombre de Columnes i Files quan s'han seleccionat múltiples Zones Horàries.
- Inclou la teva pròpia Etiqueta per a cada Zona Horària seleccionada.
- Opcionalment selecciona per Ressaltar.

- Configuracions d'aparença personalitzables.
- Configuracions personalitzades addicionals per al Rellotge Analògic per mostrar agulla de segons, mostrar passos, rellotge digital interior per exemple.
