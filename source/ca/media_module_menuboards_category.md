# Taulers de Menú: Categoria

Mostra dades per Categories contingudes en un Tauler de Menú en qualsevol lloc d'un Disseny usant **Elements**.

{feat}Menu Board: Category|v4{/feat}

El Widget de Taulers de Menú: Categoria consisteix principalment en una font de Tauler de Menú que alimenta els Elements configurats.

{tip}
Els [Taulers de Menú](media_menuboards.html) han de crear-se amb Categories definides abans d'afegir el Widget de Taulers de Menú: Categoria als Dissenys.
{/tip}

## Elements de Categoria

Els Elements estan disponibles per selecció al afegir un Widget de Taulers de Menú: Categoria a un [Disseny](layouts_editor.html) per donar als Usuaris més control sobre quins components de les Categories de Tauler de Menú usar i on es poden col·locar.

![Category Elements](img/v4_media_modules_category_elements.png)

Cada Element té un conjunt d'opcions de configuració en el Panell de Propietats. S'ha de seleccionar un Tauler de Menú per usar com a font de dades des de la pestanya **Configurar** per a cada Element usat en el Disseny. Controla com els articles han de ciclar-se especificant una [Ranura de Dades](layouts_editor.html#content-data-slots) per usar per a cadascun dels Elements afegits. Els Elements de Dades poden complementar-se encara més afegint Elements Globals per afegir formes i text que es poden posar en un Grup d'Elements per a una configuració i posicionament més fàcils.

## Descripció General

- Actualitza Elements amb noves dades de Categoria editant les dades subjacents del [Tauler de Menú](media_menuboards.html#content-create-and-configure-categories).
- Actualitza el contingut del Widget de Taulers de Menú: Categoria sense accedir als Dissenys.
- El contingut per a aquest mitjà és emmagatzemat en memòria cau pels Reproductors per reproducció fora de línia.
- Usa-ho amb el Widget [Taulers de Menú: Productes](media_module_menuboards_products.html) per crear fàcilment un menú/catàleg d'articles perquè la teva Categoria es mostri en Pantalles.
