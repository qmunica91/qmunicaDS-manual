## Alerta de Emergencia

Ayuda a garantizar la seguridad del personal, alumnos y el público mostrando alertas y mensajes de emergencia utilizando datos de feeds del Protocolo de Alerta Común (CAP) para aumentar la efectividad de las advertencias a través de tu red de Pantallas [[PRODUCTNAME]].

![Emergency_Alerts_Elements](img/media_module_emergency_alerts_elements.png)

## Descripción General de Características

- **Conector CAP** utiliza datos del servicio de feed del Protocolo de Alerta Común.
- Proporciona una **URL CAP** para extraer datos de alerta de feeds CAP.
- Añade de una variedad de **Elementos** para adaptar representaciones visuales en el editor de Diseños para el máximo impacto.
- Envía alertas de emergencia solo a **Pantallas** específicas del área que coincidan para una mensajería dirigida efectiva.
- Usa filtros estándar de la industria para adaptar la mensajería de los datos devueltos.
- Muestra múltiples mensajes de alerta de emergencia con ciclado de datos de feeds CAP.
- Establece **Criterios** al **Programar** para determinar si la alerta debe incluirse en el Bucle de Programación para Reproductores soportados.
